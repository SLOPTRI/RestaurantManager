# RestaurantManager

**RestaurantManager** is a desktop application designed to efficiently manage restaurants. It allows you to manage products, orders, and tables, providing functionalities to create, edit, and delete records easily and intuitively.

---

## Features

- **Product Management**
    - Create, edit, and delete menu items.
    - Price control.

- **Table Management**
    - Create, edit, and delete restaurant tables.
    - View the status of each table (Orders and Price).

- **Order Management**
    - Register new orders associated with tables.
    - Modify order contents.
    - Close orders and generate summaries.

- **User-friendly Interface**
    - Intuitive desktop application to streamline restaurant workflows.

---

## Technologies

- Programming language: *Java*
- Interface framework: *JavaFX*
- Data persistence using XML.
